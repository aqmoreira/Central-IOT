Com objetivo de realizar os testes e simulação das diversas situações foi desenvolvido um simulador em HTML5.
A implementação faz comunicação com um servidor Mqtt, simulando a movimentação do VLT pelo cruzamento.
Para iniciar o simulador, realize o download dos arquivos da pasta e abra no Browser o arquivo index.html
